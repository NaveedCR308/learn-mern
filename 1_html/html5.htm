Understanding HTML5 Semantics

July 23, 2024

Introduction

HTML5 introduced several semantic elements that help to define the structure of a web page.

Examples of Semantic Elements

header
The header of a section or page.
nav
The navigation links for a page.
main
The main content of the page.
aside
Content related to the main content.
section
A section of related content.
article
                            An independent piece of content.

Sample Code

                    
                        <header>
                        <h1>My Website</h1>
                        <nav>
                            <ul>
                                <li><a href="#home">Home</a></li>
                                <li><a href="#about">About</a></li>
                                <li><a href="#contact">Contact</a></li>
                            </ul>
                        </nav>
                        </header>

HTML5 Elements Comparison

Comparison of HTML5 Semantic Elements
Element Description Example Usage
<header> The header of a section or page <header>...</header>
Typically contains introductory content or navigation links
<nav> Contains the navigation links for the website
<main> The main content of the page <main>...</main>
<aside> Holds content related to the main content
Often used for sidebars
<section> Defines a section in a document <section>...</section>
Used for grouping related content
<article> Defines an independent, self-contained content
